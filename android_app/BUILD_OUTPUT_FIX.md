# Build Output 탭이 안 보일 때 해결 방법

## 방법 1: 메뉴에서 강제로 열기

1. **View** → **Tool Windows** → **Build** 클릭
   - 또는 **View** → **Tool Windows** → **Build Output** 클릭

2. 단축키 사용:
   - `Alt + 0` (영문 숫자 0) 누르기
   - 그러면 Tool Windows 목록이 나타남
   - `Build Output` 선택

## 방법 2: 빌드 실행 후 자동으로 열리게 하기

1. **Build** → **Rebuild Project** 클릭
2. 빌드가 시작되면 오른쪽 하단에 알림이 나타남
3. 알림을 클릭하면 Build Output이 열립니다

## 방법 3: 이벤트 로그에서 확인

1. Android Studio 우측 하단의 **벨 아이콘** 클릭 (알림 아이콘)
2. 최근 빌드 결과 확인

## 방법 4: Problems 탭 확인

1. 하단에 **Problems** 탭이 있는지 확인
2. 여기서 컴파일 오류를 볼 수 있습니다

## 방법 5: 상태 표시줄 확인

1. Android Studio **맨 아래 상태 표시줄** 확인
2. 빌드 결과가 간단히 표시됩니다
